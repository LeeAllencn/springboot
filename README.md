"# springboot"

# 项目基础结构

```markdown
-- springboot							# 项目根目录
  -- DB									# 数据库脚本文件目录
  -- springboot-api-service				# 页面api服务
  -- springboot-common					# 全局公共包目录
  -- springboot-external-client			# 调用外部服务客户端配置目录
  -- springboot-openapi-service			# openapi服务
  -- springboot-parent					# 项目父目录：做jar包统一管理
  -- .gitignore							# git忽略文件配置文件
  -- pom.xml							# 项目模块管理文件
  -- README.md							# 项目说明文件
```
